### What I'm working on

**[FindUtils](https://findutils.com)** — A suite of 334+ free online tools for developers, designers, and professionals. Everything runs client-side. No accounts, no ads, no data collection.

- **[Tool API](https://findutils.com/en/developers/tool-api)** — 50 tools available as a REST API at `api.findutils.com`
- **[MCP Server](https://github.com/olgunozoktas/findutils-mcp)** — 52 tools accessible from Claude, ChatGPT, and any MCP-compatible client
- **[Desktop App](https://findutils.com)** — Native desktop app built with Tauri

The stack: Astro 6, React 19, TypeScript, Tailwind CSS 4, Cloudflare Workers, Tauri (Rust).
